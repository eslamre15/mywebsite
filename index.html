About Me

I am a senior communication and electronics engineering student, working on a project focused on **cloud-based embedded malware analysis**. My project leverages a **Raspberry Pi 4B** to monitor Linux devices, analyze logs, and detect threats using AI/ML.

Project Overview

This project aims to develop a secure and efficient system for monitoring Linux devices by deploying an agent that collects logs and forwards them to a Raspberry Pi for initial threat detection. The relevant data is then sent to the cloud for further analysis.
